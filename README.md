Just my wlogout theme incase you're interested &lt;3

No install script or anything fancy, CSS was the only part of this that I actually know, but on the bright side, if a noob like me can figure it out then you probably can too. 

Either clone the repo and move the files to your .config/wlogout/ or wherever your keep your wlogout config files, or just copy and past their contence into your existing files and that should be it! 

Background not included, icons don't belong to me and the code comes from here there and everywhere, I found looking at other ppls configs on github useful which is mainly why I'm doing this, so hopefully you will too <3
